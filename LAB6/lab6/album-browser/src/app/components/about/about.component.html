About Album Browser

📖 About This Application

Album Browser is a single-page application (SPA) built with Angular that lets you browse, view, edit, and manage photo albums. It demonstrates Angular's core features including routing, HTTP client, services, and reactive data handling with Observables.

🛠️ Tech Stack

  • Framework: Angular 17+ (Standalone Components)
  • HTTP: Angular HttpClient + JSONPlaceholder REST API
  • Routing: Angular Router with route parameters
  • Reactive: RxJS Observables + async pipe
  • Language: TypeScript

📚 Course Information

  • Course: Web Development
  • Lab: Lab 6 — Routing & HTTP
  • API: JSONPlaceholder (https://jsonplaceholder.typicode.com)

✨ Features

  • Browse 100 albums from JSONPlaceholder API
  • View album details and edit titles
  • Browse photo grids for each album
  • Delete albums from the list
  • Responsive design with loading states
